# [`llama-cpp-node`](https://github.com/withcatai/node-llama-cpp)
This is an extension of the [`@llama-cpp-node/linux-x64-cuda`](https://www.npmjs.com/package/@llama-cpp-node/linux-x64-cuda) package
that provides a prebuilt binary for [`llama-cpp-node`](https://github.com/withcatai/node-llama-cpp) for Linux x64 with CUDA support.

Do not install this package directly.
