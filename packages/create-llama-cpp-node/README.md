# `create-llama-cpp-node`
## Scaffold a new [`llama-cpp-node`](https://www.npmjs.com/package/llama-cpp-node) project from a template
```bash
npm create llama-cpp-node@latest
```

And then follow the prompts.

You can directly specify the project name you want to use via the command line:
```bash
npm create llama-cpp-node@latest my-project
```
